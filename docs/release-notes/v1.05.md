# Domain AI v1.05

**Local-first AI chat for Android. Your words stay on your device.**

Domain AI answers every message on-device by default, with zero network access. The
cloud is strictly opt-in — gated by a network kill switch and a redactor that strips
sensitive information from your prompt before it's sent to a cloud model.

## What's new since 1.01

- **Faster generation on more devices.** The number of inference threads now adapts
  to your phone's performance cores instead of a fixed default — quicker on
  flagships, and it stops dragging in slow cores (and the UI) on budget phones.
- **Downloads are their own process.** Getting a model shows real progress
  ("Downloading X — 42%" / "Importing X…"), separate from loading it into memory, and
  a failed download no longer disrupts the model you're currently using.
- **Consistent model selection.** The top-bar subtitle, the chat quick-panel
  checkmark, and Settings' "In use" always agree on the active model — even mid-load.
- Polish under the hood: all on-screen text is sourced from resources, and the docs
  were brought up to date.

## Highlights

- **Local by default** — replies generated on-device; no network unless you opt in
  per message.
- **Network kill switch** — outbound requests made impossible at one auditable
  chokepoint. On by default.
- **PII redaction** — emails, phone numbers, SSNs, cards and IPs are stripped before
  any cloud call, and each cloud reply shows the exact redacted text that was sent.
- **Opt-in cloud** — connect an OpenAI-compatible provider (e.g. OpenRouter). Off by
  default; entirely your choice, per message.
- **Routing transparency** — every reply is badged On-device, Cloud, or Blocked.
- **Encrypted at rest** — conversations are stored in an AES-256 file keyed by the
  Android Keystore; backups are disabled, and there are no analytics or trackers
  (crash reporting is opt-in and off by default).
- **GPU acceleration** — optional GPU offload with an in-app CPU-vs-GPU benchmark.
- **Configurable context window** — Auto or a fixed size (bounded by your device's
  RAM), from Settings or the chat quick-panel.
- **Reliable history** — conversations persist correctly across app restarts.

## Requirements

- Android 8.0 (API 26) or newer.
- An arm64-v8a device — the native runtime is arm64-only (universal among modern
  phones; older 32-bit-only devices are not supported).
- A GGUF model: download one in-app or import your own.

## Install

Download the APK below and sideload it (enable "Install unknown apps" for your
browser or file manager if prompted). The build is signed, so future updates with
the same signature install over the top.
