# Domain AI v1.01

**Local-first AI chat for Android. Your words stay on your device.**

Domain AI answers every message on-device by default, with zero network access. The
cloud is strictly opt-in — gated by a network kill switch and a redactor that strips
sensitive information from your prompt before it's sent to a cloud model.

## Highlights

- **Local by default** — replies are generated on-device; no network is touched
  unless you opt in per message.
- **Network kill switch** — one switch makes outbound requests impossible, enforced
  at a single auditable chokepoint. On by default.
- **PII redaction** — emails, phone numbers, SSNs, cards and IPs are stripped
  before any cloud call, and each cloud reply shows the exact redacted text that
  was sent.
- **Opt-in cloud** — connect an OpenAI-compatible provider (e.g. OpenRouter) for
  cloud replies. Off by default; entirely your choice, per message.
- **Routing transparency** — every reply is badged On-device, Cloud, or Blocked, so
  the data path is never ambiguous.
- **Encrypted at rest** — conversations are stored in an AES-256 file keyed by the
  Android Keystore; backups are disabled, and there are no analytics or trackers
  (crash reporting is opt-in and off by default).
- **GPU acceleration** — optional GPU offload with an in-app CPU-vs-GPU benchmark.
- **Configurable context window** — Auto or a fixed size (bounded by your device's
  RAM), from Settings or the chat quick-panel.
- **Reliable history** — conversations persist correctly across app restarts.

## Requirements

- Android 8.0 (API 26) or newer.
- An arm64-v8a device — the native runtime is arm64-only (universal among modern
  phones; older 32-bit-only devices are not supported).
- A GGUF model: download one in-app or import your own.

## Install

Download the APK below and sideload it (enable "Install unknown apps" for your
browser or file manager if prompted). The build is signed, so future updates with
the same signature install over the top.
